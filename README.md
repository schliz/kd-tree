# Simple k-d-Tree Implementation

How to use:

1. add Maven Repository

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

2. add this project as dependency

```xml
<dependency>
    <groupId>com.github.schliz</groupId>
    <artifactId>kd-tree</artifactId>
    <version>1.0</version>
</dependency>
```
